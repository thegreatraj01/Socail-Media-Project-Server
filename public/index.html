Run API Endpoints with Postman

Click the button below to fork the Postman collection and explore the API endpoints.

Steps to explore the API in Postman:

  1. Go to the collection named: Complete Backend(Node.js) In User folder
  2. Navigate to the register request and send it to create a new user (you don't need to fill in specific details right now).
  3. Next, go to the login request and send it to log in.
  4. Once you are logged in (you might receive a token or session), you can then navigate to and send any other API requests within the collection.
